# Politique de Confidentialité

Madame, Monsieur,

DFX AG (ci-après dénommée DFX) prend très au sérieux la protection de vos données personnelles. La protection des données constitue une priorité majeure pour DFX.

Le traitement confidentiel et responsable des données personnelles issues de nos relations avec les clients, les partenaires commerciaux, les employés, les candidats et les fournisseurs constitue le fondement de notre succès commercial. Les données personnelles désignent toute information se rapportant à une personne identifiée ou identifiable.

Par conséquent, nous respectons les principes suivants lors du traitement des données personnelles :

* Confidentialité et transparence
* Respect de nos directives internes et de la politique de confidentialité
* Respect de toutes les lois et réglementations applicables

## 2. Informations générales sur la protection des données

DFX traite vos données personnelles de manière confidentielle et conformément aux dispositions légales en matière de protection des données, notamment la Loi fédérale suisse sur la protection des données (LPD) et la présente politique de confidentialité.

L'utilisation de notre site web est généralement possible sans fournir de données personnelles. Toutefois, si une personne concernée souhaite utiliser des services particuliers de notre entreprise via notre site web, le traitement de données personnelles peut s'avérer nécessaire. Si le traitement de données personnelles est nécessaire et qu'il n'existe aucun fondement juridique pour un tel traitement, nous obtenons le consentement de la personne concernée sous une forme appropriée après avoir fourni les informations adéquates.

Nous tenons à souligner que, malgré les précautions de sécurité que nous avons prises, la transmission de données sur Internet (par exemple, lors de communications par courrier électronique) peut présenter des failles de sécurité. Une protection complète des données contre l'accès par des tiers n'est pas possible.

En tant que personne (physique) concernée, il est dans votre intérêt personnel de protéger le ou les systèmes que vous utilisez (PC, ordinateur portable, etc.) contre tout accès non autorisé par des tiers, de les munir d'une protection par mot de passe adéquate et de ne pas divulguer le mot de passe à des tiers. Il est recommandé d'installer un programme antivirus disponible dans le commerce et de le mettre à jour régulièrement.

### Catégories de données personnelles

Dans le cadre de nos relations commerciales et de l'utilisation de nos services, nous traitons diverses catégories de données personnelles. Celles-ci comprennent, entre autres :

* Informations de contact : par exemple, nom, adresse, numéro de téléphone, adresse électronique.
* Données techniques : par exemple, adresses IP, informations sur l'appareil, type de navigateur.
* Données de paiement et financières : par exemple, coordonnées bancaires, données de carte de crédit, historique des transactions, déclarations fiscales.
* Données de communication : par exemple, contenu des courriers électroniques, formulaires de contact, demandes de renseignements.
* Données d'utilisation : par exemple, pages visitées, données de connexion, comportement d'utilisation sur notre site web.
* Données contractuelles : par exemple, produits achetés, services, conditions contractuelles.
* Données sensibles (le cas échéant) : par exemple, données de santé, données biométriques à des fins d'identification (uniquement avec consentement explicite), documents d'identification officiels.

La collecte de ces données est effectuée exclusivement aux fins énoncées dans la présente politique de confidentialité et dans le respect des réglementations applicables en matière de protection des données.
